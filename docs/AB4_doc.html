AB4

Propagates the state vector forward one time step using the Adams-Bashforth 4th-order method.

Syntax

F = AB4(f,t,F,h)

Description

F = AB4(f,t,y,h) updates the matrix, F, for the next sample time, given the matrix at the current time t, the function f(t,y) defining the ODE , and the step size h.
