odefun_mat2vec

Transforms a matrix-valued ODE into a vector-valued ODE.

Back to ODE Solver Toolbox Contents.

Syntax

dydt = odefun_mat2vec(F,t,y)
dydt = odefun_mat2vec(F,t,y,p)

Description

dydt = odefun_mat2vec(F,t,y) transforms the matrix-valued ODE dM/dt = F(t,M) (where F:R×R(p×p)->R(p×p) into the vector-valued ODE dy/dt = f(t,y) (where f:R×R(p^2)->R(p^2)). It is assumed that M is a square matrix.

dydt = odefun_mat2vec(F,t,y,p) transforms the matrix-valued ODE dM/dt = F(t,M) (where F:R×R(p×q)->R(p×q) into the vector-valued ODE dy/dt = f(t,y) (where f:R×Rpq->Rpq). p specifies the number of rows of M.
