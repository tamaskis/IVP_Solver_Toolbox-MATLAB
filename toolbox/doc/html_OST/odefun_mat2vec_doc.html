odefun_mat2vec

Transforms a matrix-valued ODE into a vector-valued ODE.

Syntax

Description

f = odefun_mat2vec(F) transforms the matrix-valued ODE (where into the vector-valued ODE (where ). It is assumed that is a square matrix.

f = odefun_mat2vec(F,p) transforms the matrix-valued ODE (where ) into the vector-valued ODE (where ). p specifies the number of rows of M.
