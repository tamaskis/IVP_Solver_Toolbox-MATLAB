odeIC_mat2vec

Transforms the initial condition for a matrix-valued ODE into the initial condition for the corresponding vector-valued ODE.

Syntax

y0 = odeIC_mat2vec(M0)

Description

y0 = odeIC_mat2vec(M0) transforms the initial condition for a matrix-valued ODE into the initial condition for the corresponding vector-valued ODE.

Input/Output Parameters

Variable Symbol Description Format
Input M0 initial condition for matrix-valued ODE p×q
double
Output y0 initial condition for corresponding vector-valued ODE pq×1
double
