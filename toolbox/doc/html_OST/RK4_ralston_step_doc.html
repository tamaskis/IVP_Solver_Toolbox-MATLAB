RK4_ralston_step

Propagates the state vector forward one time step using Ralston's fourth-order method (Runge-Kutta fourth-order method).

Syntax

y_next = RK4_ralston_step(f,t,y,h)

Description

y_next = RK4_ralston_step(f,t,y,h) returns the state vector at the next sample time, y_next, given the current state vector y at time t, the function f(t,y) defining the ODE , and the step size h.

Example

Consider the initial value problem

Find the solution until using RK4_ralston_step. Then, compare your result to the solution found by RK4_ralston.

First, let's define our ODE () and initial condition in MATLAB.

f = @(t,y) y;
y2 = 3;

Let's define a time vector between and with a spacing of .

h = 0.01;
t = (2:h:10)';

Solving for using RK4_ralston_step and comparing the result to the result obtained using RK4_ralston,

% preallocate vector to store solution
y = zeros(size(t));

% store initial condition
y(1) = y2;

% solving using "RK4_ralston_step"
for i = 1:(length(t)-1)
    y(i+1) = RK4_ralston_step(f,t(i),y(i),h);
end

% solving using "RK4_ralston"
[t_RK4_ralston,y_RK4_ralston] = RK4_ralston(f,[2,10],y2,h);

% maximum absolute error between the two results
max(abs(y_RK4_ralston-y))

ans =

     0

As expected, the two methods obtain identical results.
